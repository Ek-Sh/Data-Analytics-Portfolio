# Data_Analytics-Portfolio

---

Привет! Я — аналитик данных.

В этом репозитории собраны мои учебные и pet-проекты по аналитике данных, продуктовой аналитике, A/B-тестированию, BI-аналитике и работе с данными в Python, SQL, PySpark, Superset и Yandex DataLens.

Проекты демонстрируют навыки полного аналитического цикла: от предобработки данных и исследовательского анализа до проверки гипотез, построения дашбордов, формулирования выводов и подготовки рекомендаций для бизнеса.

*Общее количество проектов в репозитории:* **22**
