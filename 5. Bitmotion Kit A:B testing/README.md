# Анализ результатов A/B-тестирования интернет-магазина BitMotion Kit


***СТАТУС:*** **Завершён**


## Цель проекта:

Оценить корректность проведения A/B-теста и проверить гипотезу:

1. Упрощение интерфейса интернет-магазина увеличит конверсию зарегистрированных пользователей в покупку.
2. В течение семи дней после регистрации конверсия пользователей в покупку вырастет минимум на 3 процентных пункта.
3. Новая версия сайта покажет статистически значимый прирост конверсии по сравнению с контрольной группой.


## Описание проекта:

Интернет-магазин BitMotion Kit продаёт геймифицированные товары для людей, ведущих здоровый образ жизни. Среди популярных товаров магазина — эспандер со счётчиком и напоминанием, а также подстольный велотренажёр с Bluetooth.

По отзывам пользователей, текущий интерфейс сайта оказался слишком сложным. Чтобы повысить удобство онлайн-магазина, привлечь новых клиентов и увеличить количество покупок, компания разработала новую версию сайта и протестировала её на части пользователей.

В проекте проводится оценка корректности A/B-теста `interface_eu_test`: проверяется качество данных, распределение пользователей по группам, пересечения с другими тестами, наличие дубликатов и соответствие горизонта анализа техническому заданию.

После подготовки данных рассчитывается конверсия пользователей в покупку в течение первых семи дней после регистрации и проводится статистическая проверка различий между группами A и B.


## Вывод исследования:

1. В тесте `interface_eu_test` участвовали две группы: A — контрольная группа, B — группа с новым интерфейсом.
2. Распределение пользователей между группами было близким к равномерному.
3. Пересечений пользователей между группами одного теста не выявлено.
4. Были обнаружены пользователи, участвующие более чем в одном тесте. Для корректности анализа такие пользователи были исключены.
5. После очистки в анализе осталось 9 963 пользователя.
6. Конверсия в покупку в группе A составила 27,81%.
7. Конверсия в покупку в группе B составила 29,54%.
8. Абсолютный прирост конверсии в тестовой группе составил около 1,73 процентного пункта.
9. Односторонний z-тест для двух пропорций показал статистически значимое различие между группами: p-value = 0,028.
10. Новая версия интерфейса положительно повлияла на конверсию, однако целевой прирост в 3 процентных пункта достигнут не был.


## Использование результатов исследования:

Результаты исследования могут быть использованы интернет-магазином BitMotion Kit для принятия решения о дальнейшем развитии интерфейса сайта.

Несмотря на статистически значимый рост конверсии, эффект оказался ниже ожидаемого. Поэтому перед полным внедрением новой версии интерфейса рекомендуется дополнительно оценить бизнес-эффект: прирост выручки, изменение среднего чека, окупаемость разработки и влияние интерфейса на промежуточные этапы пользовательской воронки.


## Технические особенности проекта:

В ходе исследования была проведена проверка качества данных: изучены типы данных, пропуски, явные дубликаты и структура таблиц.

В таблице событий были выявлены и удалены полные дубликаты. Также были обнаружены пользователи, участвующие более чем в одном тесте. Для исключения влияния конкурирующего эксперимента такие пользователи были удалены из анализа.

Для оценки результата теста был ограничен горизонт анализа: учитывались только события, совершённые пользователями в течение первых семи дней после регистрации.

Целевой метрикой стала конверсия зарегистрированных пользователей в покупку. Для проверки статистической значимости различий между группами использовался односторонний z-тест для двух пропорций.
