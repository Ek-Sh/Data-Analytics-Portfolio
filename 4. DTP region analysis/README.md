# Анализ факторов ДТП па основе проекта "Карта ДТП"


***СТАТУС:*** **Завершён**

## Цель проекта:

На основе данных некоммерческого проекта «Карта ДТП» проверить качество данных о дорожно-транспортных происшествиях и изучить основные закономерности аварийности:

1. Проверить наличие пропусков в данных.
2. Проверить наличие явных и неявных дубликатов.
3. Оценить корректность типов данных.
4. Определить, как менялось число ДТП по дням недели и месяцам.
5. Сравнить число ДТП для групп водителей с разным стажем.
6. Подготовить рекомендации по улучшению качества сбора данных.

##Описание проекта:

В проекте анализируются данные «Карты ДТП» — некоммерческого проекта, посвящённого проблеме дорожно-транспортных происшествий в России.

Данные содержат информацию о ДТП в Кировской и Московской областях: координаты происшествий, дату и время, погодные условия, освещённость, состояние дорожного покрытия, количество участников и пострадавших.

Дополнительно используются таблицы с информацией об участниках ДТП и транспортных средствах. Эти данные позволяют изучить аварийность в разрезе водительского стажа и оценить полноту собранной информации.

Основной акцент проекта сделан на проверке качества данных, поиске ошибок и подготовке рекомендаций для заказчика.

##Вывод исследования:

1. В данных встречаются пропуски, особенно в полях, связанных с адресом, схемой ДТП, координатами и водительским стажем.

2. Явные дубликаты в данных не обнаружены.

3. Идентификаторы ДТП являются уникальными, что позволяет корректно связывать таблицы между собой.

4. В отдельных столбцах используются неоптимальные типы данных, которые можно изменить для удобства анализа и экономии памяти.

5. Число ДТП различается по дням недели: больше происшествий происходит ближе к выходным и в выходные дни.

6. По месяцам наблюдается сезонность: количество ДТП увеличивается в тёплые месяцы.

7. Среди участников ДТП чаще встречаются водители с небольшим стажем, однако для точной оценки риска необходимо учитывать общее количество водителей в каждой стажевой группе.

8. Для повышения качества дальнейшего анализа заказчику важно улучшить полноту заполнения ключевых полей.

## Использование результатов исследования:

Результаты исследования могут быть использованы командой проекта «Карта ДТП» для улучшения качества сбора данных и повышения точности дальнейшего анализа аварийности.

Проверка пропусков, дубликатов и типов данных помогает выявить слабые места в процессе заполнения информации о ДТП.

Анализ распределения ДТП по времени и водительскому стажу может быть полезен для разработки мер по повышению безопасности на дорогах, планирования профилактических мероприятий и выявления групп повышенного риска.

## Технические особенности проекта:

* таблицы с данными о ДТП;
* таблицы с информацией об участниках ДТП;
* таблицы с информацией о транспортных средствах.

* переименование столбцов для удобства работы;

* анализ и оптимизация типов данных;
