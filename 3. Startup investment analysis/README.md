# Исследование стартапов и инвестиционной активности

***СТАТУС***: **Завершён**

##Цель проекта:

На основе данных о стартапах, финансировании и возвратах инвестиций проанализировать рынок и сформировать инвестиционную рекомендацию на 2015 год.
В рамках проекта необходимо:

1. Изучить данные о стартапах и финансировании.
2. Провести предобработку данных.
3. Разделить компании по срокам привлечения инвестиций.
4. Выделить массовые, средние и нишевые сегменты рынка.
5. Найти компании с аномальным объёмом финансирования.
6. Проанализировать типы финансирования.
7. Изучить динамику инвестиций по годам.
8. Определить перспективные отрасли для инвестирования.

## Описание проекта:

В проекте исследуются данные о стартапах, которые привлекали инвестиции в разные годы. Датасет содержит сведения о компаниях, рыночных сегментах, датах финансирования, количестве раундов, общем объёме привлечённых средств и типах финансирования.
Дополнительно используется датасет с данными о возвратах средств по годам и типам финансирования.
Исследование выполняется с позиции 2015 года: необходимо определить перспективные отрасли и наиболее подходящий тип финансирования.

## Вывод исследования:

1. Данные требовали предобработки: были выявлены пропуски, некорректные типы данных и неинформативные строки.

2. После очистки было удалено около 8,94% строк, оставшегося объёма данных достаточно для анализа.

3. Большинство компаний получили финансирование один раз, но основной объём инвестиций сосредоточен у компаний с длительным финансированием.

4. Рынок стартапов неоднороден: небольшое число компаний привлекает очень крупные суммы.

5. Высокие типичные объёмы финансирования характерны для clean technology и biotechnology.

6. В 2014 году рост показали сегменты apps, medical devices, technology, transportation и частично internet.

7. Среди типов финансирования лидирует venture — по объёму инвестиций и числу компаний.

8. Seed и debt_financing используются часто, но обычно связаны с меньшими суммами.

9. Доля возвратов средств остаётся низкой, что типично для венчурного рынка.

##Использование результатов исследования:

Результаты могут быть использованы инвесторами и аналитиками для выбора перспективных отраслей и оценки инвестиционной стратегии.
Исследование помогает определить растущие сегменты, популярные типы финансирования, объём рынка и возможные инвестиционные риски.

##Технические особенности проекта:

В ходе проекта выполнены:
* загрузка и изучение данных;
* обработка пропусков, дубликатов и типов данных;
* очистка столбца funding_total_usd;
* преобразование дат;
* заполнение пропусков в mid_funding_at;
* классификация компаний по срокам финансирования;
* выделение массовых, средних и нишевых сегментов;
* поиск выбросов методом IQR;
* анализ типов финансирования;
* анализ динамики инвестиций по годам;
* расчёт среднего объёма одного раунда;
* анализ возвратов средств;
* подготовка выводов и рекомендаций.
