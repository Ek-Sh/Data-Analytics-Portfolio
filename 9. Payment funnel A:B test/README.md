# Анализ A/B-теста новой платёжной воронки интернет-магазина

***СТАТУС:*** **Завершён**

## Цель проекта:

На основе результатов A/B-теста оценить эффективность новой платёжной воронки интернет-магазина и проверить гипотезу о том, что её внедрение увеличивает конверсию пользователей в покупку.

В рамках проекта необходимо:

1. Проверить корректность проведения A/B-теста.
2. Оценить распределение пользователей между тестовыми группами.
3. Сравнить конверсию пользователей в покупку в группах A и B.
4. Проанализировать кумулятивную динамику ключевых метрик.
5. Проверить статистическую значимость различий между группами.
6. Сформулировать рекомендации по результатам эксперимента.

## Описание проекта:

В распоряжении находятся данные крупного интернет-магазина, работающего преимущественно в Азии. Магазин провёл A/B-тест новой платёжной воронки (`new_payment_funnel_test_summer_2025`), целью которого было повышение конверсии пользователей в покупку.

В ходе проекта выполнены загрузка и предобработка данных о новых пользователях, их сессиях, покупках и участниках эксперимента. Проведена проверка качества данных, исследовано распределение пользователей по группам, рассчитаны ключевые показатели эффективности, построены графики кумулятивной динамики конверсии и покупок. Для оценки результатов эксперимента применён статистический тест, позволивший определить наличие или отсутствие статистически значимых различий между тестовыми группами.

## Вывод исследования:

1. Выполнена проверка корректности проведения A/B-теста и соответствия эксперимента техническому заданию.
2. Проанализировано распределение пользователей между контрольной и тестовой группами, выявлены возможные особенности выборок.
3. Рассчитана и сравнена конверсия пользователей в покупку для обеих групп эксперимента.
4. Исследована кумулятивная динамика покупок и основных метрик на протяжении всего периода тестирования.
5. Проведена статистическая проверка гипотезы о различии конверсии между группами.
6. Сформулирован вывод об эффективности новой платёжной воронки и подготовлены рекомендации по дальнейшим действиям.

## Использование результатов исследования:

Полученные результаты могут быть использованы продуктовой командой при принятии решения о внедрении новой платёжной воронки, доработке пользовательского сценария оформления заказа и планировании последующих A/B-тестов.

## Технические особенности проекта:

Перед проведением анализа выполнена предобработка данных: проверены типы данных, обработаны пропуски и дубликаты, объединены данные из нескольких источников.

Для анализа были рассчитаны показатели конверсии, исследована кумулятивная динамика метрик, выполнена визуализация результатов эксперимента и проведена статистическая проверка гипотезы с использованием подходящего критерия для сравнения двух независимых выборок.
