## Цель проекта:

Разработать интерактивный BI-дашборд в Apache Superset для анализа игрового рынка и поведения игроков.

В рамках проекта необходимо отследить:
1. Динамику цен на игры.
2. Распределение установок по играм, жанрам и платформам.
3. Активность игроков через получение достижений.
4. Популярность игровых жанров и платформ.
5. Структуру релизов по платформам, разработчикам и издателям.


## Описание проекта:

Компания, специализирующаяся на разработке игр, поставила задачу создать аналитический дашборд для отслеживания ключевых трендов игрового рынка.

Для анализа были предоставлены данные за период с 2007 по 2025 год. Данные содержат информацию об играх, платформах, жанрах, разработчиках, издателях, игроках, установках, достижениях и изменении цен.

В проекте был создан единый датасет в Superset, объединяющий несколько таблиц: `Games`, `Prices`, `Players`, `Players_games`, `Achievements` и `History`. Для сохранения полной информации об играх использовались `LEFT JOIN`, чтобы не потерять игры без связанных данных по установкам, ценам или достижениям.

На основе подготовленного датасета был разработан дашборд с четырьмя тематическими страницами:

1. **Игровые достижения** — анализ количества достижений, редких достижений и их распределения по жанрам, играм и платформам.
2. **Анализ игроков** — изучение количества игроков, установок, распределения пользователей по странам и платформам.
3. **Стоимость игр** — анализ средней и медианной стоимости игр, динамики цен и изменения цен по играм.
4. **Анализ игр** — исследование релизов, жанров, платформ, разработчиков и издателей.


## Вывод исследования:

1. Основная активность игроков сосредоточена на современных платформах PS4 и PS5. Эти платформы лидируют по количеству установок, игроков и релизов.

2. Наибольший пользовательский интерес вызывают жанры Action, Action-Adventure, Shooter и Platformer. Они входят в число лидеров по количеству установок.

3. По странам наибольшая доля игроков приходится на США. Также заметную долю занимают Испания, Великобритания, Франция, Бразилия и Германия.

4. Анализ достижений показывает, что их количество и редкость можно использовать как дополнительный показатель вовлечённости игроков.

5. Наиболее активное развитие каталога игр наблюдается на PS4 и PS5. Старые платформы, такие как PS3 и PS Vita, имеют значительно меньшую долю релизов и установок.

6. Средняя и медианная стоимость игр различается по жанрам, что может быть полезно для анализа ценовой политики и позиционирования продуктов.

7. Динамика цен по платформам позволяет отслеживать изменения стоимости игр во времени и выявлять периоды роста или снижения цен.


## Использование результатов исследования:

Результаты проекта могут использоваться топ-менеджерами и аналитиками игровой компании для оценки состояния игрового рынка, анализа популярности платформ и жанров, планирования новых релизов и корректировки ценовой политики.

Дашборд помогает быстро находить точки роста, оценивать пользовательский интерес, анализировать вовлечённость игроков и принимать решения по продвижению игр.


## Технические особенности проекта:

Для проекта был создан единый аналитический датасет в Superset, объединяющий данные из нескольких связанных таблиц.

В дашборде настроены фильтры по периоду, гранулярности, платформе, жанру, разработчику, издателю, названию игры и поддерживаемым языкам.

Также была реализована кросс-фильтрация по платформам и использована логика Jinja для анализа изменения количества игроков год к году.
