# Анализ продаж медицинских пробирок для компании «ЛайфЛайн»

## Цель проекта:

Разработать интерактивный дашборд для анализа продаж медицинских пробирок компании «ЛайфЛайн» и обеспечить пользователей инструментом для оперативного контроля ключевых показателей отдела продаж.

Дашборд должен помочь:

1. Отслеживать сумму продаж, количество проданных товаров и среднюю стоимость товара.
2. Анализировать динамику продаж по месяцам и кварталам.
3. Изучать продажи в разрезе регионов, поставщиков, категорий товаров, типов клиентов и каналов продаж.
4. Сравнивать показатели за разные периоды и выявлять критичные изменения.


## Описание проекта:

Компания «ЛайфЛайн» занимается продажей медицинских пробирок различного назначения: для анализа крови, хранения мазков ПЦР и других лабораторных целей.

В рамках проекта были собраны и уточнены бизнес-требования к дашборду. Исходные требования заказчика были сформулированы в общем виде, поэтому дополнительно определены ключевые метрики, пользовательские сценарии, фильтры и структура аналитических вкладок.

На основе требований разработан дашборд в DataLens. Он предназначен для директора отдела продаж, менеджеров по продажам и менеджеров по работе с поставщиками.

Дашборд включает обзор ключевых показателей, анализ динамики продаж, географию продаж, рейтинг товаров и категорий, структуру продаж по поставщикам и сравнение показателей за выбранные периоды.


## Вывод исследования:

1. Общая сумма продаж за анализируемый период составила **325,14 млрд руб.**
2. Количество проданных товаров составило **39,88 млн единиц**.
3. Средняя стоимость товара составила около **8,15 тыс. руб.**
4. Продажи представлены в **21 регионе**.
5. Динамика суммы продаж в целом стабильная, однако по отдельным периодам наблюдаются колебания.
6. Наибольший вклад в продажи вносят стерильные, клинические, лабораторные и специализированные пробирки.
7. В структуре продаж заметную роль играют разные типы клиентов: аптеки, медицинские учреждения и частные лица.
8. По отдельным поставщикам, каналам продаж и типам клиентов зафиксированы отрицательные изменения относительно предыдущего периода.


## Использование результатов исследования:

Дашборд может использоваться директором отдела продаж для контроля общих показателей и оценки динамики бизнеса.

Менеджеры по продажам могут использовать дашборд для анализа продаж по регионам, каналам и типам клиентов.

Менеджеры по работе с поставщиками могут отслеживать структуру продаж поставщиков, выявлять снижение показателей и анализировать проблемные сегменты.

Результаты проекта помогают быстрее находить зоны роста, контролировать выполнение продаж и принимать управленческие решения на основе данных.


## Технические особенности проекта:

Данные представлены в реляционной базе PostgreSQL и содержат информацию о продажах, товарах, поставщиках, клиентах, каналах продаж и регионах.

В ходе проекта выполнены:

- сбор и документирование бизнес-требований;
- определение ключевых метрик отдела продаж;
- проектирование структуры дашборда;
- создание обзорной вкладки с основными показателями;
- настройка фильтров по дате, региону, поставщику, категории товара, типу клиента и каналу продаж;
- визуализация динамики суммы продаж и средней стоимости товара;
- построение карты географии продаж;
- создание рейтинга товаров и категорий;
- анализ связи между количеством проданных товаров и суммой продаж;
- визуализация структуры продаж поставщиков по типам клиентов;
- разработка вкладки детализации продаж;
