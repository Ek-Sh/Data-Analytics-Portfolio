# Анализ благотворителей фонда «АиФ. Доброе сердце»

***СТАТУС:*** **Завершён**

## Цель проекта:

На основе данных благотворительного фонда «АиФ. Доброе сердце» исследовать поведение благотворителей и рассчитать ключевые метрики:

1. Ежемесячную динамику активных благотворителей — MAU.
2. Retention Rate благотворителей по когортам в зависимости от пола.
3. Средний чек и количество пожертвований мужчин и женщин.
4. LTV благотворителей из Москвы, Санкт-Петербурга и Екатеринбурга.

## Описание проекта:

Фонд «АиФ. Доброе сердце» помогает людям со сложными диагнозами: оплачивает операции, лекарства и медицинское оборудование, а также оказывает юридическую и психологическую поддержку подопечным.

В проекте анализируются данные о благотворителях, пожертвованиях и действиях пользователей. Фонду важно лучше понимать поведение доноров, чтобы эффективнее выстраивать коммуникацию, повышать повторные пожертвования и развивать долгосрочные отношения с благотворителями.

В рамках проекта был разработан интерактивный дашборд в Yandex DataLens. На дашборде визуализированы основные продуктовые и бизнес-метрики: MAU, Retention Rate, средний чек и LTV по ключевым когортам.

## Вывод исследования:

1. MAU демонстрирует общий рост с отдельными пиками и спадами. Повышение активности может быть связано с сезонными благотворительными кампаниями, медийными активностями фонда или внешними инфоповодами.
2. Retention Rate благотворителей быстро снижается после первого месяца. Это говорит о том, что значительная часть пользователей после первого пожертвования не возвращается к повторным платежам.
3. Женщины показывают более высокий уровень удержания по сравнению с мужчинами. Это может указывать на большую лояльность женской аудитории и её склонность к регулярной поддержке фонда.
4. Женщины совершают больше пожертвований, однако их средний чек ниже. Мужчины, наоборот, делают пожертвования реже, но на более крупные суммы.
5. Среди трёх крупных городов наибольший LTV наблюдается у благотворителей из Екатеринбурга, несмотря на меньшее количество доноров. Москва лидирует по количеству благотворителей.

## Использование результатов исследования:

Результаты исследования могут быть использованы фондом для улучшения коммуникационной стратегии, повышения повторных пожертвований и развития региональных кампаний.

На основе полученных выводов фонд может:

1. Настроить цепочки коммуникаций после первого пожертвования.
2. Развивать персонализированные рассылки для разных групп благотворителей.
3. Усилить работу с лояльной женской аудиторией.
4. Отдельно протестировать коммуникации для мужчин с акцентом на крупные разовые пожертвования.
5. Использовать региональные различия для планирования кампаний в Москве, Санкт-Петербурге и Екатеринбурге.

## Технические особенности проекта:

В данных присутствовали особенности, которые необходимо было учитывать при расчётах. Поле с датой регистрации благотворителя также могло содержать дату редактирования профиля, поэтому оно не подходило для точного определения новых пользователей.

Для расчёта Retention Rate была выбрана когорта благотворителей, совершивших пожертвование в августе 2022 года. Благотворители без информации о поле были исключены из анализа.

При расчёте метрик учитывались только успешно оплаченные пожертвования. Заказы с другими статусами были исключены, так как они могли искажать показатели MAU, среднего чека, Retention Rate и LTV.

Для визуализации результатов был разработан дашборд в Yandex DataLens. Основные графики дашборда отражают динамику активных пользователей, удержание по полу, средний чек и LTV по регионам.
