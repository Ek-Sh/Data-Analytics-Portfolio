# Дашборд эффективности рекламных кампаний мобильного приложения


***СТАТУС:*** **Завершён**

## Цель проекта:

На основе данных мобильного приложения создать интерактивный дашборд для анализа эффективности рекламных кампаний, активности пользователей и их конверсии в покупки.

В рамках проекта необходимо оценить:
1. Динамику выручки, затрат и ROI.
2. Эффективность рекламных кампаний по типам рекламы.
3. Результативность рекламных каналов.
4. Поведение пользователей в зависимости от типа устройства.
5. Конверсию пользователей из сессий в заказы.


## Описание проекта:

Компания разрабатывает мобильное приложение и использует разные рекламные каналы для привлечения пользователей. Для оценки эффективности маркетинга были использованы данные о рекламных расходах, пользовательских сессиях и заказах.

В проекте были созданы виртуальные датасеты на основе таблиц `pa_costs`, `pa_sessions` и `pa_orders`. Эти данные были объединены и использованы для расчёта ключевых бизнес-метрик: выручки, затрат, ROI, количества заказов, конверсии в заказ, количества сессий и средней длительности сессии.

На основе подготовленных данных был построен интерактивный дашборд с KPI-блоком, графиками по типам рекламы, рекламным каналам и устройствам пользователей.


## Вывод исследования:

1. За анализируемый период выручка составила **30 878.1**, а количество заказов — **5 788**.
2. ROI составил **95.9%**, что говорит о положительной окупаемости рекламных вложений в целом.
3. Конверсия в заказ составила **19.9%**, то есть примерно каждый пятый пользователь совершал покупку после сессии.
4. Количество сессий составило **29 106**, средняя длительность сессии — **29.9 минуты**.
5. По динамике ROI видно, что эффективность рекламных кампаний менялась по месяцам: часть направлений показывала отрицательную окупаемость, но к концу периода некоторые типы рекламы вышли в положительную зону.
6. Наибольшая доля сессий приходится на пользователей **iPhone**.
7. Не все рекламные каналы с высоким количеством сессий показывают высокую окупаемость, поэтому оценивать эффективность рекламы нужно не только по трафику, но и по выручке, затратам и ROI.


## Использование результатов исследования:

Результаты проекта могут быть использованы маркетинговой командой для оценки эффективности рекламных кампаний, перераспределения рекламного бюджета и поиска наиболее прибыльных каналов привлечения пользователей.

Дашборд позволяет заказчику регулярно отслеживать ключевые показатели, сравнивать каналы между собой и оперативно принимать решения по оптимизации маркетинговых расходов.


## Технические особенности проекта:

Для анализа были использованы три исходные таблицы:

- `pa_costs` — данные о рекламных расходах;
- `pa_sessions` — данные о пользовательских сессиях;
- `pa_orders` — данные о заказах и выручке.

В ходе проекта были созданы виртуальные датасеты:

- `sessions` — для анализа пользовательской активности;
- `orders` — для расчёта заказов и выручки;
- `costs` — для анализа рекламных затрат.


Дашборд состоит из трёх основных аналитических разделов:

1. **По типу рекламы** — анализ ROI и баланса доходов и расходов.
2. **По каналам рекламы** — анализ ROI, количества сессий и средней длительности сессии.
3. **По устройствам** — анализ распределения и динамики сессий по типам устройств.
