## Цель проекта:

Разработать интерактивный дашборд в Yandex DataLens для анализа матчей DOTA 2, который позволяет оценивать игровые показатели, популярность героев и предметов, баланс фракций и эффективность различных игровых ролей.

## Техническое задание:

Дашборд должен интерактивно отображать следующую информацию:

* общее количество матчей;
* количество игроков;
* количество героев;
* соотношение побед между фракциями Radiant и Dire;
* распределение игроков по типу атаки (Melee / Ranged);
* топ-10 героев по количеству участий в матчах;
* топ-10 предметов, приобретаемых первыми;
* распределение матчей по длительности и количеству игроков;
* сравнение характеристик героев (атака, скорость передвижения, восстановление здоровья, убийства и опыт в минуту) у победителей и проигравших в зависимости от основной роли.

## Описание проекта:

Исходные данные представлены тремя CSV-файлами:

* matches.csv — информация о матчах;
* players.csv — статистика игроков, выбранные герои и приобретённые предметы;
* heroes.csv — характеристики игровых персонажей.

На этапе подготовки данных были проверены исходные таблицы, настроены связи между ними по идентификаторам матча и героя, а также созданы необходимые вычисляемые поля для анализа.

Дашборд объединяет основные игровые показатели в одном окне и позволяет интерактивно исследовать данные с помощью фильтров.

Для визуализации использованы:

* индикаторы (KPI);
* круговые диаграммы;
* комбинированная диаграмма;
* горизонтальные столбчатые диаграммы;
* сводные таблицы;
* интерактивные селекторы.

## Использование результатов:

Разработанный дашборд может использоваться командой разработки DOTA 2 для:

* мониторинга игрового баланса между фракциями;
* анализа популярности героев и игровых предметов;
* исследования поведения игроков;
* оценки эффективности различных игровых ролей;
* поиска направлений для балансировки персонажей и игровой механики.
